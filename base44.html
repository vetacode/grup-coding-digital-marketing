Pakai ChatGPT generate code tapi tidak ngerti cara kerjanya. Saat error → stuck, saat deploy → gagal.

Belajar HTML, CSS, JS tapi belum pernah bikin landing page utuh. Tidak punya portfolio & tidak percaya diri.
